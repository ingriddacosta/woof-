# Petshop Web Application

## Overview
This project is a simple web application for a pet shop, providing information about services, products, and location. The application is designed to be user-friendly and informative, catering to pet owners looking for services and products for their pets.

## Project Structure
The project consists of the following main files:
- `index.html`: The homepage of the pet shop.
- `location.html`: A page detailing the location of the pet shop.
- `products.html`: A page showcasing the products available for pets.
- `services.html`: A page outlining the services offered by the pet shop.

## Features
- **Responsive Design**: The application is designed to be responsive and accessible on various devices.
- **User-Friendly Navigation**: Easy navigation between different sections of the website.

## Getting Started
To run this project locally:
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Open the `index.html` file in your web browser.
